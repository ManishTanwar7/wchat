# WhatsApp Chat Bot

Simple WhatsApp-style chat website with C++ backend.

## Files
- index.html
- style.css
- script.js
- server.cpp
- Dockerfile

## Features
- Linked list
- Classes
- Abstraction
- Inheritance
- Virtual function
- Polymorphism
- Dynamic memory allocation
- Exception handling
